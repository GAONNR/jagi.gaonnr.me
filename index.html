자기소개서 글자 수 세기

당신의 자기소개서 글자 수(엄밀히 말하면 Byte수)를 세어줍니다.

Author's Github Download .exe Author's Blog

프로그램 소개

이 프로그램은 자기소개서의 글자 수를 세어 주는 프로그램입니다. 사실은, 글자 수가 아니라 Byte수를 세어줍니다. (한글: 2Bytes/글자, 영문 및 숫자: 1Byte/글자) 이렇게 만들었던 이유는, 본인이 소속된 학교가 자기소개서 글자 수를 Bytes수로써 판별했었기 때문입니다.

사용 방법

본 프로그램의 사용 방법은 매우 간단합니다. 프로그램을 실행하면 보이는 커다란 텍스트 상자에 자기소개서를 붙여 넣으세요. 오른쪽 위의 '글자수 세기' 버튼을 클릭하면, 하단에 Byte수가 출력됩니다.

유의사항

본 프로그램은 C#을 이용하여 개발되었습니다. 실행 시 .NET FRAMEWORK 4.0 이상이 필요하다는 메시지가 출력될 수 있습니다. 설치해주세요.
또한 이 프로그램은 본인이 입시기간 때 심심한 나머지 얼렁뚱땅 만든 게 2년이 다 되어가는 놈입니다. 그래서 소스코드도 없고, 치명적인 버그가 있을지도 모릅니다. 수정은 당연히 못 합니다. 그냥 새로 개발하는 게 빨라요.
당신의 자기소개서는 절대 수집되지 않습니다. 2년 전의 저한테는 그런 발상과 기술이 없었어요. 안심하고 쓰세요.
